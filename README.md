 💰 Expense Manager

A modern desktop application built with **Python** and **CustomTkinter** to help users manage their personal expenses through an intuitive graphical interface.


✨ Features

- ✅ Register new expenses
- ✅ Edit existing expenses
- ✅ Delete expenses
- ✅ View all recorded expenses
- ✅ Expense statistics by category
- ✅ Support for multiple currencies (BOB and USD)
- ✅ Custom expense categories
- ✅ Data stored locally using JSON
- ✅ Modern graphical user interface

🛠️ Built With

- Python 3
- CustomTkinter
- JSON
- Tkinter MessageBox
- Tkinter SimpleDialog

🚀 Getting Started

 1. Clone the repository

```bash
git clone https://github.com/DilanRiv/Gestionador_de_Gastos-Expense_Manager.git
```

2. Navigate to the project

```bash
cd Gestionador_de_Gastos-Expense_Manager
```

 3. Install CustomTkinter

```bash
pip install customtkinter
```

4. Run the application

```bash
python app_gastos.py
```
